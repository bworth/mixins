    height

    @mixin height($from, $to: null) { ... }

    Description

    Simple height breakpoint mixin for responsive layouts. Values are pixel based, since screens are pixel based, and the upper value is reduced by one pixel to avoid clashes. If the lower value is 0 or null then an upper value must be specified.

    Parameters

    $from

    The lower value of the matching height, 0 or null if there is no non-zero lower value

    		Number— none

    true-vh

    @mixin true-vh($property, $vh-height, $height-delta: null) { ... }

    Description

    iOS reports the same viewport height in vh units whether Safari UI elements are visible or not. To fix this we create a more accurate CSS custom property, set via JavaScript.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $property

    Height property to be used (height, min-height, padding-block-start etc.)

    		String— none
    $vh-height

    Required vh height value (unitless)

    		Number— none
    $height-delta

    Any additional non-vh height value to be added or subtracted

    		Numbernull

    Example

    Set the CSS custom property with this companion script

    document.documentElement.style.setProperty('--vhUnit', window.innerHeight / 100 + 'px');

    Using this mixin to set a min-height of 80vh - 64px

    @include true-vh(min-height, 80, -64px);

    Author

    • Ben Worthington

    position

    @mixin position($top: 0, $right: $top, $bottom: $top, $left: $right) { ... }

    Description

    Position mixin, sets all the position properties. Follows the same top, right, bottom, left order as other layout properties. Shorthands can also be used, e.g. one value equates to the same value for all, two values equate to vertical and horizontal values, and three values equates to top, horizontal and bottom values.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $top

    Value for the top property

    		Number0
    $right

    Value for the right property

    		Number$top
    $bottom

    Value for the bottom property

    		Number$top
    $left

    Value for the left property

    		Number$right

    Used by

    • [mixin] absolute
    • [mixin] fixed
    • [mixin] relative
    • [mixin] sticky

    • Ben Worthington

    pointer-triangle

    @mixin pointer-triangle($direction, $size, $color: currentcolor, $border-width: 0, $border-color: currentcolor) { ... }

    Description

    Creates a simple CSS pointer shape (triangle) without having to use images or SVG. The pointer is placed on the container based on the supplied $direction, e.g. on the top of the container if $direction is up, and is centered on the container in the other dimension.
